# Guides

## Build

```sh
pnpm install
pnpm run dev
```

## Run in Browser

For simple ui testing, just open the url in browser.

For test as real extension.

1. Build the extension.

    ```sh
        pnpm run build
    ```

2. Open Browser.
3. Go to extension page from settings.
4. Enable developer mode.
5. Load unpacked extension.
