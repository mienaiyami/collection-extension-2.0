Privacy Policy

Last Updated: March 2024

Data Collection and Storage

  • All collection data is stored locally on your device using browser's storage API
  • No personal data is collected or stored on external servers
  • Your data remains private and under your control
  • You can export, import, or delete your data at any time

Google Drive Integration

  • Uses Google OAuth 2.0 for secure authentication
  • Access is limited to application data folder only
  • Cannot access your other Google Drive files
  • Authentication tokens are stored locally on your device
  • Token refresh operations are handled securely via Cloudflare Worker
  • Sync is optional and can be disabled at any time
  • You can revoke access through your Google Account settings

Browser Permissions

  • Storage: Required to save your collections locally
  • Tabs & ActiveTab: Required to manage and interact with browser tabs
  • Identity: Required for Google OAuth authentication
  • SidePanel: Required for extension interface in browser side panel
  • Incognito: Optional permission for incognito mode support

Data Security

  • All communication with Google Drive uses secure HTTPS
  • No analytics or tracking code is included
  • Extension is open source - code can be audited on GitHub
  • Regular security updates through browser stores

Your Rights

  • Access your data anytime through the extension interface
  • Delete all local data through extension settings
  • Disable Google Drive sync at any time
  • Uninstall the extension to remove all local data
